# elasticesearch-kibana-install-using-docker
엘라스틱서치 실습을 위해 도커로 엘라스틱서치, 키바나를 설치
[ksundong/docker-elk-kor](https://github.com/ksundong/docker-elk-kor) 저장소의 내용을 참조하였으며, logstash를 제외하였습니다.



## docker-compose up

```bash
 $> docker-compose up -d
```
